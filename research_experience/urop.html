TECAN to ECHO converter:

May 2017 – Oct 2017

Abstract:

The full research project is in a Github directory, private, by request.

Bioinformatics is the interdisciplinary study of computer science, mathematics, and molecular biology, which requires programming, mathematical modeling, and experimental analysis. Sponsored by CIDAR Lab and STEM Pathways, the TECAN Freedom 150 to ECHO 550 conversion uses bioinformatics to automate script conversions between these two robots. Currently, the CIDAR Lab uses the MoClo Assembly bio tool to generate TECAN scripts in order to automate robot pipetting; however, the tool lacks the capability of generating other robot scripts. The ECHO robot has several advantages over the TECAN robot. For example, the TECAN robot requires higher resource consumption than the considerably more efficient ECHO robot. Written in the Python programming language, the conversion tool reads the TECAN script in a text file and writes the corresponding ECHO script in an Excel CSV file. The tool is interchangeable; it converts the ECHO script back to TECAN. Should an error occur during the liquid handling process, the conversion tool has an email system to inform the researcher of such error. The Python conversion tool will allow researchers to have flexibility. Since the CIDAR Lab requires a tool that will increase variability with automated scripting, the conversion tool will be crucial to maintaining sound research in synthetic biology for the future.

General description of project:

I implemented a Python program to convert input TECAN script into corresponding ECHO output script. Within the program, we made a design choice to have an email system that notified researcher quickly to reduce error in conversion. The program managed to reduce resource consumption by about 50%, which led to significant research efficiency.
